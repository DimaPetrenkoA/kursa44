Назначение программы:
Программа предназначена для моделирования процесса внесения данных о летательных аппаратах в базу.
Условия выполнения программы:
Для выполнения программы необходима ПЭВМ типа IBM PC или программно совместимая с ней, функционирующая под управлением операционной системы семейства Windows (не ниже Windows XP). 
Выполнение программы:
Для переключения языка пользователь должен выбрать соответствующий пункт меню, используя стрелки вверх, вниз и клавишу Enter.
Для возврата в предыдущее меню используется соответствующий пункт меню.

Описание программы 
Общие сведения:
Программа Fly.exe предназначена для моделирования работы авиационного бюро. 
Функциональное назначение:
Программа предназначена для использования в высших учебных заведениях с целью демонстрации главных концепций объектно-ориентированного подхода при разработке ПО средствами языка С++.
Используемые технические средства:
Для запуска программы требуется персональный компьютер с частотой процессора  не  менее  1.2 ГГц  и  объемом  оперативной  памяти  не  менее 256 Мб.
Вызов и загрузка:
Вызов и загрузка программы осуществляется путем запуска исполняемого файла Fly.exe.
Входные данные:
В качестве исходных данных программа использует данные вводимые пользователем с клавиатуры, а также данные выгруженные из кеша.
Выходные данные:
Выходные данные выводятся на экран и файлы которые прописаны в коде программы.
